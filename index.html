☁️ クラウド同期 (GitHub Gist)

GitHub Gistを使ってポートフォリオデータをクラウドに保存し、複数デバイスで共有できます。

  1. GitHubでPersonal Access Tokenを作成 → gist スコープのみ選択してください
  2. 生成されたトークンを下の欄に貼り付け
  3. 「接続」をクリック → 以降は自動保存されます
トークンはこのデバイスのみに保存されます（外部サーバーへは送信されません）

📋 スプレッドシートからインポート

スプレッドシートのセル範囲をコピー Ctrl+C して、下のエリアに貼り付けてください。
1行目はヘッダー行として認識します。 列の順序は問いません。

自動認識する列名の例:
ティッカー/銘柄コード/証券コード  ・  銘柄名/会社名  ・  保有数量/株数/口数  ・  平均取得単価/取得単価
時価・簿価・損益・増減率などの計算列は自動スキップされます。
